## Proyectos

**[DeepRacer STEAM Agent](https://github.com/carlonox/Deepracer-STEAM-Agent)**
Agente de IA (Hermes) que controla un AWS DeepRacer físico mediante SSH sobre Tailscale. Incluye sistema RAG con 20 manuales técnicos, reconocimiento ArUco, control por voz y una interfaz web con React y Express.
`Docker` `ROS2` `Python` `React` `Express.js` `OpenCV`

**[Traffic Sign Recognition](https://github.com/carlonox/traffic-sign-recognition-ai)**
Sistema de reconocimiento de señales de tránsito en tiempo real mediante CNN entrenada con un dataset de 43 clases. Detección por webcam con OpenCV y clasificación con Keras/TensorFlow.
`Python` `TensorFlow` `OpenCV` `Keras`

**[StudentRegistry](https://github.com/carlonox/StudentRegistry)**
Aplicación de escritorio para gestión académica con CRUD completo, sistema de admisión automática basado en puntajes y persistencia en MySQL. Arquitectura MVC.
`Java` `Swing` `MySQL`

**[Challup](https://github.com/shofi1003/BugBusters_grupo11_ingesoft1)**
Aplicación móvil de retos diarios personalizados con IA. Perfilación inteligente mediante cuestionario, generación de retos ajustados al progreso del usuario y gamificación con rachas, puntos y rankings. Proyecto grupal para Ingeniería de Software I (Grupo BugBusters, 2025-2).
`Kotlin` `Firebase` `Android` `IA`

**[Triqui en C++](https://github.com/carlonox/Tictactoe-en-CPP)**
Tres en Raya con CPU que evalúa 6 niveles de estrategia: ganar, bloquear, maximizar jugada propia, tomar la mejor del oponente, priorizar esquina o aleatorizar.
`C++`
